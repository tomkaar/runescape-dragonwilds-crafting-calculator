# RuneScape Dragonwilds: Crafting Calculator

This tool helps players determine the materials needed to craft various items within the game.

## Get started

### 1. Clone the repository:

```bash
git clone git@github.com:tomkaar/runescape-dragonwilds-crafting-calculator.git
```

### 2. Navigate to the project directory:

```bash
cd runescape-dragonwilds-crafting-calculator
```

### 3. Install dependencies:

This project is using PNPM as the package manager. If you don't have it installed, you can do so by following the instructions at <https://pnpm.io/installation>.

```bash
pnpm install
```

### 4. Sync data:

Make sure to fetch and parse the latest data by running the following commands:

```bash
pnpm fetch-data
pnpm parse-data
```

### 5. Start the development server:

```bash
pnpm dev
```
